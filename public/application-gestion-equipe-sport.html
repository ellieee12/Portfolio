Application de Gestion de Biens Immobiliers

Takraw Go! est une application web destinée à la gestion d'une équipe de sport, spécifiquement pour le jeu de takraw. Le takraw, ou sepak takraw, est un sport proche du volley-ball et très populaire en Asie du Sud-Est. Contrairement au volley-ball, ce sport se joue avec les pieds à l'aide d'un ballon en rotin ou en plastique, opposant deux équipes de trois joueurs. Cette application permet de gérer les joueurs d'une équipe de takraw ainsi que les matchs passés et à venir. Un tableau de bord est également développé pour résumer les statistiques des performances des joueurs.
Le site est accessible en utilisant "admin" comme l'identifiant et le mot de passe.

Architecture et méthodologie

Architecture MVC

L'architecture MVC est implémentée dans le cadre du développement web, ce qui permet de structurer le projet de manière modulaire et extensible. Cela facilite la maintenance et l'évolution du projet, notamment pour les futures fonctionnalités. Cette organisation sera particulièrement utile par la suite, lors de l'implémentation d'une API pour récupérer des données depuis le serveur PHP.

Patron DAO

Le patron DAO rend le code plus modulaire, lisible et facile à maintenir. En séparant la logique de la base de données de celle de l'application, il permet de faire évoluer l'application sans impacter le code lié à la gestion des données.
